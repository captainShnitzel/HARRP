# External references

External papers, manufacturer datasheets, standards, and copyrighted course material are intentionally not committed to the repository.

Maintain bibliographic entries and official links here. Store personal working copies outside Git or in the ignored `references/private/` directory.

Initial references used during HARRP V1 development include:

1. Mick et al., “Reachy, a 3D-Printed Human-Like Robotic Arm as a Testbed for Human-Robot Control Strategies,” *Frontiers in Neurorobotics*, 2019.
2. Nurpeissova, Tursynbekov and Shintemirov, “An Open-Source Mechanical Design of ALARIS Hand: A 6-DOF Anthropomorphic Robotic Hand,” IEEE ICRA, 2021.
3. Spectra Symbol flex-sensor datasheet.
4. ROBOTIS Dynamixel actuator and protocol documentation.
5. Microcontroller, motor-driver and IMU manufacturer documentation.
